Syllabus

Prerequisite: Basic Computer Knowledge, Networks, Database Management Systems.

Course Objectives:

  • To build strong skills in Frontend and Backend Web Technologies
  • To learn how modern websites and web apps are built
  • To understand security, optimization, and database interaction

Modules:

Module Content Hours
1. INTRODUCTION TO HTML & CSS Basics of HTML, CSS styling, Document structure, Forms, Lists 5
2. ADVANCED HTML5 New elements, Audio/Video integration, Canvas, Local Storage 5
3. XML & JSON Data Representation, XML Structure, JSON Syntax, Parsing JSON 5
4. JAVASCRIPT FUNDAMENTALS DOM Manipulation, Event Handling, Functions, Objects 5
5. PHP & SERVER-SIDE PROGRAMMING PHP Syntax, Sessions, Forms, File Handling 5
6. MYSQL DATABASE INTEGRATION Database Connection, CRUD Operations, Queries 5
7. INTRODUCTION TO NODE.JS Server-side JavaScript, Creating Servers, NPM Modules 5
8. API CREATION & CONSUMPTION RESTful APIs, Fetch API, Axios, Authentication Basics 5
9. RESPONSIVE WEB DESIGN Media Queries, Mobile-first Approach, Frameworks (Bootstrap) 5
10. WEB SECURITY BASICS OWASP Top 10, SQL Injection, Cross-Site Scripting, HTTPS 5

Total: 50 Hours

Lab Manuals

Sample HTML Lab Programs:

Program 1: Basic HTML Page

<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a simple HTML page.</p>
</body>
</html>

Program 2: HTML Table

<!DOCTYPE html>
<html>
<head>
    <title>Simple Table</title>
</head>
<body>
    <table border="1">
        <tr><th>Name</th><th>Age</th></tr>
        <tr><td>John</td><td>25</td></tr>
        <tr><td>Jane</td><td>22</td></tr>
    </table>
</body>
</html>

Program 3: HTML Form

<!DOCTYPE html>
<html>
<head>
    <title>Simple Form</title>
</head>
<body>
    <form>
        Name: <input type="text" name="name"><br><br>
        Email: <input type="email" name="email"><br><br>
        <input type="submit" value="Submit">
    </form>
</body>
</html>
